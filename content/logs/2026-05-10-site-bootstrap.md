## 今日目标

完成个人技术站的最小可用版本，优先支持写作、预览和 GitHub Pages 部署。

## 关键决策

- 使用 Hugo 作为静态站点生成器。
- 使用 PaperMod 作为主题。
- 第一阶段部署到 GitHub Pages，不使用 VPS。
- 仓库名使用 `blog`，因此默认访问路径是 `https://tuyh3.github.io/blog/`。
- 不引入 Node.js 或 npm 依赖。

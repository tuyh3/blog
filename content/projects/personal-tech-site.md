## 项目定位

这是一个低维护成本的个人技术博客 / 作品集网站，用于沉淀公开技术信用资产。

## 技术栈

- Hugo
- PaperMod
- Markdown
- GitHub Pages

## 当前状态

第一版目标是完成站点结构、写作模板、示例内容和自动部署流程。
